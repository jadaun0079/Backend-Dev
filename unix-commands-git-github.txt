File & Directory
pwd          → current directory
ls           → list files
ls -a        → include hidden files
ls -l        → detailed view
cd folder    → move into folder
cd ..        → move back
mkdir test   → create folder
rmdir test   → delete empty folder
rm file.txt  → delete file
rm -r dir    → delete folder forcefully

File Handling

touch a.txt          → create file
cat a.txt            → show file content
nano a.txt / vim a.txt → edit file
cp a.txt b.txt       → copy
mv a.txt dir/        → move / rename

git config --global user.name "Jatin"
git config --global user.email "email@gmail.com"

Start Git

git init                 → initialize repo
git status               → check status
git add .                → stage all files
git commit -m "msg"      → save changes

Branching


git branch               → list branches
git branch dev           → create branch
git checkout dev         → switch branch
git checkout -b feature  → create + switch
git merge dev            → merge branch

Connect Repo

git remote add origin <repo-url>
git remote -v

Push Code

git push origin main
git push -u origin main   → first time

Pull / Clone

git clone <repo-url>
git pull origin main
